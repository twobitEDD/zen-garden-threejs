# Zen Sacred Geometry SVG Pack

This pack contains Japanese-inspired sacred geometry SVG files for a Three.js project.

## SVG files

- seed_of_life_zen.svg
- flower_of_life_lite_zen.svg
- enso_mandala.svg
- shippo_tsunagi.svg
- seigaiha_circles.svg
- kiku_rosette.svg

## Recommended Three.js import

```js
import { SVGLoader } from 'three/examples/jsm/loaders/SVGLoader.js';
```

A helper file is included:

```js
loadZenSvgExample.js
```

## Suggested uses

1. Visual overlay above sand.
2. Rake animation path guide.
3. Occupancy/nav-map seed for preferred or blocked pattern zones.
4. Decal-style reference layer that can be hidden once the rake has drawn the pattern.
